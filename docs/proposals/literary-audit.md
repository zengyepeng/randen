# 燃灯 — 文学性审计系统

> 设计日期: 2026-07-19 | 目标: 为 37 维技术审计补充独立的文学质量维度 | 优先级: P1

---

## 问题定义

当前 37 维审稿覆盖了情节逻辑、角色行为、世界规则、视角纪律等技术层，保证了小说"不犯低级错误"，但无法回答：**这章好看吗？**

技术审计能发现"伏笔 f042 已 3 卷未提及"，但无法发现"这一章所有人物说话都像一个模子里刻出来的"。

关系：**技术审计在先（先修 bug），文学审计在后（再提升品质）**。

---

## 一、十大文学性审计维度

每个维度独立评分 1-10，总分 100。扣分 2+ 必须附带修改建议。

### 1. 节奏张力
情绪是否有起伏？高潮点是否合理（太早→后面乏力，太晚→冲击透支，没有→扣 3+）？

```yaml
# 检查点
- tension_curve: 本章是否有波峰波谷？
- climax_position: 高潮点是否在合理位置？
- section_pacing: 段落间速度切换是否自然？
- breather_pattern: 紧张后是否有呼吸空间？
```

**示范修法** — 问题："前 60% 都在对话谈事情，后 40% 突然打斗" → 建议："在第 4-5 段插入一个短促的动作片段打断对话节奏" → 示范：*"她正要开口解释，门被推开了。一个小弟子跌跌撞撞冲进来，脸色煞白——她没有说完。"*

### 2. 留白艺术
有没有"说太透"（"他感到非常愤怒" → 扣分；"他的手在桌下攥成了拳" → 加分）？伏笔暗示是否变成了明示？

**示范修法** — 问题："林月看到顾恒身上的血迹，她感到一阵心痛" → 建议："删掉心理描写，血迹本身就是最好的叙事" → 示范：*"林月看着顾恒衣襟上那片暗褐色。他别过头去。'不碍事。'"*

### 3. 意象密度
本章有没有能让人记住的画面？是否只用了视觉（调动触觉/听觉/嗅觉/温度）？是否在用"月光如水""寒风刺骨"套话？

**示范修法** — 问题："她感到悲伤。秋天的景色让她想起了过去" → 示范：*"渡口的芦苇全白了。风过时，芦花碎成细末，落在她肩上，像很多年前那场没有等到人的雪。"*

### 4. 对话潜文本
人物说的和他们真正想表达的之间有没有张力？对话是否停留在"交换信息"层面？每个人说话腔调是否可区分？

**示范修法** — 问题：A:"你还记得青云峰的约定吗？" B:"当然记得。" → 示范：*A:"青云峰。你还记得？" B:"那破地方。早忘了。" A:"" B:"什么时候走？"*（读者懂：他没忘）

### 5. 情感递进
情感变化是否有层次？从 A 情绪到 B 情绪的转换有没有触发点？高潮时刻的爆发是否被前面的铺垫足够支撑？

**示范修法** — 问题："刚才还在闲聊，下一段突然翻脸决裂" → 建议："在闲聊到翻脸之间加一个触发点——一句话触碰逆鳞，或一个眼神的解读偏差"

### 6. 语言质地
有没有金句？句子是否经过打磨？动词是否精准？句式是否有变化？

**示范修法** — 问题："他的手里有一把很长的剑，剑是银色的，剑柄上刻着龙" → 示范：*"他手里那把剑，从鞘口到剑尖，银白一条。剑柄上的龙纹，被血浸过，看不清了。"*

### 7. 视角纪律
第三人称有无"窜视角"（同一段落进入多个人物内心）？叙述者是否知道了视角角色不该知道的信息？

**示范修法** — 问题："顾恒推开门，松了口气——他不知道林月就在窗外的树上" → 建议：拆成两个段落 + 明确标记 [顾恒视角] / [林月视角]

### 8. 信息释放时机
该藏的藏够了吗？该给的信息给出了吗？信息释放是自然的还是"突然天降答案"？

### 9. 重复与变奏
前文的标志性意象/动作/台词在本章是否有呼应？重复是有意义的变奏（强化主题）还是词汇贫乏？

**示范修法** — 问题："前文'雨中的渡口'是核心意象，本章写城市场景完全没联系" → 示范：*"城门口挤满了逃难的人。灰尘扬起，像那天下在渡口的雨。只不过这一次，没有船了。"*

### 10. 读者期待管理
章末钩子是否能推动翻页？本章开头暗示的"会有某事发生"是否兑现？反转是否有伏笔支撑？

**示范修法** — 问题："但危险正在逼近…"（空泛威胁）→ 示范：*"脚步声从走廊尽头传来，越来越近。他听过这个脚步声——在北境，那个雪夜，他以为对方已经死了。"*

```yaml
  - name: "对话潜文本"   weight: 1.0
  - name: "情感递进"     weight: 1.0
  - name: "语言质地"     weight: 1.0
  - name: "视角纪律"     weight: 1.0

  - name: "重复与变奏"   weight: 1.0

thresholds:
  excellent: 85   # 文学品质优秀
  good: 70        # 合格，需微调
  acceptable: 55  # 需修改后重审
  rewrite: 0      # 建议重写
```

### 审稿流程

```
步骤 1: 37 维技术审计 → 任何 P0/P1 错误阻断后续
步骤 2: 文学性审计（仅在步骤 1 pass 后运行）
步骤 3: 综合报告，作者看到 → 【技术: 92/100 ✅ 文学: 78/100 ⚠️】
```

### 报告格式

```markdown
# 文学性审计 — 第 0153 章 "渡口无声"
技术预审: ✅ 通过 | **总分: 72/100 — ⚠️ 微调**

| 维度 | 得分 | 状态 |
|------|------|------|
| 节奏张力 | 7/10 | ✅ |
| 留白艺术 | 5/10 | ⚠️ |
| 意象密度 | 8/10 | ✅ |
| 对话潜文本 | 6/10 | ⚠️ |
| … | … | … |

### 关键修复
1. **留白艺术 (−3)**: 第 5 段"林月知道他是在说谎…" → 删心理旁白，保留动作
2. **对话潜文本 (−2)**: 顾林对话缺潜流 → 对话内容与真实情绪错位
3. **重复与变奏 (−2)**: "雨"意象与前三章无变奏呼应
```

## 三、Python 伪代码

```python
class LiteraryAuditor:
    """文学性审计器"""
    
    def __init__(self, chapter_text: str, persona: dict = None):
        self.dimensions = {
            "rhythm_tension": {"name":"节奏张力","score":7,"issues":[]},
            "white_space": {"name":"留白艺术","score":7,"issues":[]},
            "imagery_density": {"name":"意象密度","score":7,"issues":[]},
            "dialogue_subtext": {"name":"对话潜文本","score":7,"issues":[]},
            "emotional_progression": {"name":"情感递进","score":7,"issues":[]},
            "linguistic_texture": {"name":"语言质地","score":7,"issues":[]},
            "pov_discipline": {"name":"视角纪律","score":7,"issues":[]},
            "info_release": {"name":"信息释放时机","score":7,"issues":[]},
            "repetition_variation": {"name":"重复与变奏","score":7,"issues":[]},
            "reader_expectation": {"name":"读者期待管理","score":7,"issues":[]},
        }
    
    def add_issue(self, dim: str, severity: int, desc: str, fix: str, example: str = ""):
        self.dimensions[dim]["issues"].append({
            "severity": severity, "description": desc,
            "fix_suggestion": fix, "example": example
        })
        self.dimensions[dim]["score"] = max(1, self.dimensions[dim]["score"] - severity)
    
    def run_checks(self):
        """文本级检测"""
        # 留白: 检测"感到/觉得/意识到"等心理旁白词
        feeling_kws = ["感到", "觉得", "意识到", "认为"]
        for kw in feeling_kws:
            if self.text.count(kw) > 5:
                self.add_issue("white_space", 2, f"'{kw}'出现{self.text.count(kw)}次",
                               f"将至少一半改写为动作/环境镜像")
                break
        # 意象: 检测感官描写
        sensory = ["颜色","声音","气味","温度","触感","味道","苍茫","灰","青","锈"]
        if not any(w in self.text for w in sensory):
            self.add_issue("imagery_density", 3, "缺乏感官描写",
                           "在场景转换处加一句具象感官描写")
        # 类似扩展检测器...
    
    def audit(self) -> dict:
        self.run_checks()
        total = sum(d["score"] for d in self.dimensions.values())
        overall = round(total / len(self.dimensions) * 10)
        return {"overall_score": overall, "dimensions": self.dimensions}
```
