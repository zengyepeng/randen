## 四、生成检查清单

写作完成后，用以下清单自检：

- [ ] 无禁用词/表达
- [ ] 对话占比在目标范围内（±5%）
- [ ] 段落长短有变化，无连续3个以上等长段落
- [ ] 叙述基调一致
- [ ] 无明显AI套路（参见 `craft/ai_patterns.yaml`）
- [ ] 角色声音有区分度
- [ ] 场景切换自然
